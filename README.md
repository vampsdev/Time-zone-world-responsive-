# 🌍 Fusos Horários do Mundo

> Site interativo com horários, clima e relógios em tempo real de países e estados do mundo inteiro.

🔗 **Site ao vivo:** [vampirosdev.github.io/Time-zone-world-responsive-](https://vampirosdev.github.io/Time-zone-world-responsive-/)

---

## ✨ Funcionalidades

- 🕐 Relógio digital e analógico em tempo real para cada país
- 🌡️ Temperatura e condição climática com ícones
- 🗺️ Mapa mundial interativo com modo **Satélite**
- 🌙 Botão para alternar entre **Tema Escuro** e **Tema Claro**
- 🔍 Barra de busca para encontrar qualquer país ou estado
- 🇧🇷 Aba exclusiva com todos os **27 estados do Brasil** e seus fusos
- 📱 Layout responsivo — funciona em celular e computador
- ⚡ 100% offline — sem dependências externas

## 🚀 Como usar localmente

1. Baixe o arquivo `index.html`
2. Abra no navegador (Chrome, Firefox, Edge)
3. Pronto — não precisa instalar nada!

## 🛠️ Tecnologias

- **HTML5** — estrutura
- **CSS3** — tema claro/escuro, animações
- **JavaScript** — relógios em tempo real via `Intl API`
- **SVG** — mapa mundial embutido
